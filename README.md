# palindrome

https://travis-ci.org/[edcoffin]/[palindrome].png

Test whether a word is a palindrome.

## Getting Started
Install the module with: `npm install palindrome`

```javascript
var palindrome = require('palindrome');
palindrome.awesome(); // "awesome"
```
